# MindFree

A mobile-first productivity app for daily notes, task tracking, reminders, and progress analytics.

Built with `Next.js`, `React`, `Supabase`, and a custom theme system inspired by Notion and Material Design 3.

This project is currently in active setup and feature planning. The first major implementation area is authentication, followed by the core notes, tasks, progress, reminders, and profile flows.
